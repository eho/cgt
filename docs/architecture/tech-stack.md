# Tech Stack

This project is planned as a web app.

## Chosen Tools

- Runtime and package manager: Bun
- Frontend build tool: Vite
- UI framework: React

## Package Management

Use Bun from the repository root.

- `bun install` installs dependencies.
- `bun run SCRIPT_NAME` runs package scripts.
- `bunx TOOL_NAME` runs one-off tools.

Do not create npm, Yarn, or pnpm lockfiles for this project.

## Project Commands

No package scripts exist yet. Add canonical development, test, typecheck, lint, and build commands here after the Vite React app is scaffolded.
