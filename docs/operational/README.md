# Operational

## Local Development

Install dependencies and run commands from the repository root with Bun:

```sh
bun install
bun run dev
```

The app is a client-only Vite React application. Asset and scenario data remain in browser `localStorage`.

## Verification

Use these commands before handing off calculator changes:

```sh
bun run test
bun run build
```

Use this directory for runbooks, local setup, deployment notes, CLI usage, and troubleshooting.
